# CNI-R1-Explorer
A prototype exploration of R1 institutions that are CNI members

This is a prototype tool to allow exploration of the relationships between CNI members that have been designated R1 by Carnegie Classification of Institutions of Higher Education and other details including state.

More details on the organizations can be found at https://www.cni.org/about-cni/membership/members.

Details about the Carnegie Classification methods and historical changes can be found at https://carnegieclassifications.acenet.edu/carnegie-classification/classification-methodology/2025-institutional-classification/. This prototype is not an endorsement of the methods or effects of these research and teaching classifications.

This prototype visualization was developed by pairing with Claude AI to develop html and is presented for exploration. It can be further developed with your feedback and suggestions. Please share those thoughts - and report errors - as pull requests!
